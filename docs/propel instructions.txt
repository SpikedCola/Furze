propel instructions (using example project ticketscraper)

- make table/add/change columns/etc phpmyadmin/wherever
- if just removing a column, you can jump to modifying `database\schema.xml` (delete the appropriate row). otherwise,
- `cd TicketScraper\source\classes\vendor\bin`
- run `propel database:reverse` to dump database
  - this generates `TicketScraper\source\classes\vendor\bin\generated-reversed-database\schema.xml`
- modify `TicketScraper\database\schema.xml` using `generated-reversed-database\schema.xml` as a reference
- once happy, run `propel build` to build classes using schema.xml

if the class name or namespace has changed, or a new table has been added:
- `cd TicketScraper\source\classes`
- run `composer dump-autoload` to generate autoloader related stuff (todo: not have to do this)
